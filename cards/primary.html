Newsify.work

Description

Newsify is a full-stack news aggregation REST API that collects and displays news articles from over 60,000 sources. Utilizing the GNews API, users can search, bookmark, and query articles with a variety of search parameters.

Tech Stack
  • Node.js (Express)
  • Javascript
  • CSS (Sass)
  • MongoDB
  • GNews Api (Documentation)
  • PUG (Server Side Rendering)
